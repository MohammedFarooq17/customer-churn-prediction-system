# customer-churn-prediction-system
Developed an end-to-end machine learning pipeline to predict telecom customer churn using the Telco dataset. Applied data preprocessing, feature engineering, and classification models to analyze customer behavior and improve prediction accuracy.
## ⚙️ Project Pipeline
* **Data Collection:** Importing the raw Telco dataset.
* **Preprocessing:** Handling missing values and feature scaling.
* **Exploratory Data Analysis:** Visualizing churn trends and correlations.
* **Model Building:** Training the Random Forest Classifier.
* **Evaluation:** Assessing performance using accuracy and F1-score metrics.
